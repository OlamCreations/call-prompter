SETTINGS

Ollama Setup free easy
  1. Download Ollama from ollama.ai
  2. Install and open it
  3. Open a terminal and run: ollama pull llama3
  4. Done. No API key needed.
OpenAI Setup paid
  1. Go to platform.openai.com/api-keys
  2. Sign in or create an account
  3. Click "Create new secret key"
  4. Copy the key (starts with sk-)
  5. Paste it below
Anthropic Setup paid
  1. Go to console.anthropic.com/settings/keys
  2. Sign in or create an account
  3. Click "Create Key"
  4. Copy the key (starts with sk-ant-)
  5. Paste it below
Claude Code CLI no key needed recommended
  1. Click the button below to install
  2. Open a terminal and run claude
  3. It opens your browser to log in — done
  4. No API key needed, uses your Claude account
Custom API
  1. Any OpenAI-compatible API works
  2. Enter the base URL and API key below
  3. Examples: Together AI, Groq, Fireworks, local vLLM
